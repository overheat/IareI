# Website 仓库说明

## 目录定位

本目录是 IareI 官网源码目录，对应 GitHub Pages 独立仓库，当前作为主工程下的嵌套 Git 仓库存在。

- 当前站点以静态页面为主，现有入口文件是 `index.html` 与 `styles.css`。
- 目录内保留了独立 `.git` 与远程 `origin` 配置，后续网站改动应在本目录内单独执行 `git status`、`git commit`、`git push`。
- 不要把本目录当作主仓库 `device/`、`edge/`、`cloud/` 架构的一部分实现混改；这里承载的是对外官网与项目信息展示。

## 修改规则

- 优先小步修改，保持站点可直接在浏览器打开验证。
- 不要假定这里已经接入构建系统、包管理器或 CI；若要新增，请先在文档中明确说明。
- 修改品牌文案、产品描述或型号信息时，需与主仓库文档保持一致；若不一致，先指出冲突再修改。
- `CNAME` 是 GitHub Pages 域名配置，除非明确要求，不要删除或重置。
- 若新增资源文件，优先使用相对路径，并确认 GitHub Pages 静态托管可直接访问。
